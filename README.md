CSS Modifiers

### Requirements
  - Gulp
  - Gulp-SASS
  - Gulp-Jade
  - NodeJS
  - PostCSS
  - PostCSS plugins
    - autoprefixer
    - postcss-pxtorem
    - css-mqpacker
    - postcss-custom-selectors
    - perfectionist

### Before Init

- Install NodeJS [link](https://nodejs.org/)
- install gulp global

```
npm install gulp -g
```

### Init

```
git clone https://github.com/vaeum/css.modfiles.git
cd css.modfiles
npm install
gulp
```
